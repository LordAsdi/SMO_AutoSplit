FAQ

The preview is glitching / lagging severely.

This happens when the video capture backend wasn't initialized properly. To fix this issue, restart the application.

The auto splitter isn't working reliably.

Make sure there are no overlays or borders around the gameplay. The game screen has to be fully visible and aligned pixel perfectly. The resolution should be at least 854x480p and the framerate should be 60 fps.

Text appears too large.

Right click on "SMO AutoSplit.exe". Go to Properties -> Compatability -> Change high DPI settings. Tick the box that says "Override high DPI scaling behaviour" and choose "System (Enhanced)". Click Ok.

The version number is the same after an automatic update.

If the version number in the bottom right corner doesn't increase after an automatic update you have to manually install the updated version. Download the latest version and extract the zip file into the same folder where your current installation is located.

There are a few example route files in the routes directory of SMO AutoSplit.

Getting Started

1. LiveSplit Server

Download and Install LiveSplit Server according to the readme.
Add a LiveSplit Server component to your LiveSplit Layout.
Every time you restart LiveSplit you have to enable the server by right clicking and selecting Control -> Start Server.

2. OBS Virtual Cam

You have to use the Virtual Cam plugin because the integrated Virtual Cam is not compatible with SMO AutoSplit.
Download and install the OBS Virtual Cam plugin according to the readme.
Inside OBS right click your capture card and select filters. Add a VirtualCam filter under Effect Filters. Set Buffered Frames to 0 to reduce latency and check the Keep Aspect Ratio checkbox.
Every time you restart OBS you need to click start on the VirtualCam filter.

3. SMO AutoSplit Settings

Go to the settings tab and select the OBS-Camera that you just configured.
DO NOT SELECT "OBS Virtual Camera". IT DOES NOT WORK WITH SMO AUTOSPLIT!
Restart the application for this setting to take effect.

4. SMO AutoSplit Route

To create a route, go to the route tab and select a start condition.
To add splits, click the plus icon next to the split type.
Splits can be reordered by drag and drop.
Once your route is finished, go back to the dashboard tab. The auto splitter is now ready.
You can also take a look at the example routes in the routes directory.
For further information, take a look at the Readme.
